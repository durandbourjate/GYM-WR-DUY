# Skill: Prüfungen erstellen

## Zweck
Dieses Skill beschreibt, wie schriftliche Prüfungen, Tests und Kurztests für den Unterricht in Wirtschaft und Recht erstellt werden. Es gilt für alle Gefässe (SF, EWR, EF) und alle Stufen (GYM1–GYM4).

## Wann dieses Skill anwenden
- Der Lehrer bittet um eine Prüfung, einen Test oder Kurztest.
- Der Lehrer bittet um Prüfungsfragen zu einem bestimmten Thema.
- Der Lehrer möchte eine bestehende Prüfung überarbeiten oder ergänzen.

## Vorgehen

### 1. Klärung vor der Erstellung
Folgende Angaben sind nötig (nachfragen, falls nicht angegeben):
- **Klasse und Stufe**: z.B. 28bc9f9s, GYM2
- **Gefäss**: SF, EWR oder EF
- **Fachbereich(e)**: Recht, BWL, VWL (oder Kombination)
- **Themen / behandelter Stoff**: Kapitelangaben, Lernziele oder hochgeladene Zusammenfassungen
- **Dauer**: in Minuten (z.B. 40, 45, 90 Minuten)
- **Hilfsmittel**: keine / Gesetzestexte / Taschenrechner / etc.
- **Datum der Prüfung** (für den Kopf der Prüfung)
- **Besondere Wünsche**: z.B. Schwerpunkt auf bestimmtes Thema, bestimmte Fragetypen

### 2. Stoffgrundlage
- **Prüfungsinhalte basieren ausschliesslich auf dem eingereichten Lernmaterial** (Zusammenfassungen, Lehrbuchkapitel, Lernziele, Unterrichtsnotizen). Claude erfindet keine Inhalte, die nicht im Material vorkommen.
- **Wissens- und Verständnisfragen (K1–K2)** beziehen sich direkt auf Begriffe, Definitionen, Zusammenhänge und Beispiele aus dem Unterrichtsmaterial.
- **Anwendungs-, Analyse- und Beurteilungsfragen (K3–K6)** dürfen und sollen hingegen auf **neuen, selbst formulierten Beispielen, Fallstudien und Situationen** basieren, die im Unterricht so nicht vorkamen. Der Transfer auf neue Kontexte ist Teil der Kompetenzprüfung.
- Falls kein Material hochgeladen wurde: Nachfragen, welcher Stoff behandelt wurde, bevor Fragen formuliert werden.

### 3. Umfang und Punkteverteilung
- **Faustregel**: ca. 1 Punkt pro Minute Prüfungszeit.
- Die Punkteverteilung soll die Gewichtung der Themen widerspiegeln.
- Bei jedem Aufgabenteil die Punkte angeben.

### 4. Taxonomiestufen nach Bloom
Jede Prüfung soll verschiedene kognitive Anforderungsstufen abdecken:
- **K1 Wissen**: Fakten abrufen, Definitionen nennen, Begriffe kennen.
- **K2 Verstehen**: Zusammenhänge erklären, mit eigenen Worten wiedergeben, Unterschiede erläutern.
- **K3 Anwenden**: Gelerntes auf neue Situationen übertragen, Berechnungen durchführen, Fälle lösen.
- **K4 Analysieren**: Sachverhalte zerlegen, Ursachen identifizieren, Argumente prüfen.
- **K5 Beurteilen**: Stellung nehmen, bewerten, Vor-/Nachteile abwägen.
- **K6 Gestalten**: Eigene Lösungen entwickeln, Strategien entwerfen, Konzepte erstellen.

Angestrebte Verteilung (Richtwert, je nach Stufe anpassen):
- GYM1 (Einstieg): Schwerpunkt K1–K3, wenig K4–K5
- GYM2: K1–K4, etwas K5
- GYM3/GYM4: Breite Abdeckung K1–K5, punktuell K6

Bei den Lösungen immer die Taxonomiestufe pro Teilaufgabe angeben.

### 5. Fragetypen variieren
Jede Prüfung soll **mindestens drei verschiedene Fragetypen** verwenden. Mögliche Typen:

| Fragetyp | Beschreibung | Taxonomie (typisch) |
|---|---|---|
| **Multiple Choice** | 1 richtige Antwort aus 4 Optionen (☐ Checkbox-Format) | K1–K2 |
| **Richtig/Falsch** | Aussagen beurteilen, falsche korrigieren (Tabellenformat) | K1–K2 |
| **Lückentext** | Schlüsselbegriffe einsetzen | K1 |
| **Zuordnungsaufgabe** | Begriffe/Konzepte in Tabelle zuordnen | K2 |
| **Kurze Textantwort** | 1–3 Sätze, z.B. Definition, Unterschied erklären | K1–K2 |
| **Erläuterung / Begründung** | Ausführliche Antwort, Zusammenhänge erklären | K2–K4 |
| **Fallstudie / Textbasiert** | Fragen zu einem vorgegebenen Text, Zeitungsartikel oder Grafik | K3–K5 |
| **Berechnung** | Rechnerische Aufgabe (z.B. BIP, Gini, Bilanz) | K3 |
| **Zeichnung / Visualisierung** | Diagramm, Grafik oder Schema erstellen (Zeichenfeld) | K3 |
| **Beurteilung / Stellungnahme** | Eigene Meinung formulieren und begründen | K5–K6 |
| **Fallbeispiel mit Gesetzesanwendung** (Recht) | Sachverhalt subsumieren, Gesetzesartikel anwenden | K3–K5 |

### 6. Dokumentformat (Word .docx)

#### Kopfbereich
Tabelle mit: Klasse, Name (leer), Datum, Zeit, Max. Punkte, Hilfsmittel, Erreichte Punkte (leer), Note (leer).

#### Punkteübersicht
Tabelle mit allen Aufgaben, maximalen Punkten und Feld für erreichte Punkte.

#### Antwortfelder
- **Keine Antwortlinien** (keine "___"-Linien für Schülerantworten).
- Stattdessen: **Rahmenlose Tabelle** als Platzhalter für die Schülerantwort.
- Die Tabellenhöhe entspricht dem erwarteten Platz für die Antwort.
- In dieser Tabelle steht die Lösung als **versteckter Text**.

#### Zeichenfelder
Für Zeichnungsaufgaben (z.B. Lorenzkurve, Wirtschaftskreislauf): Tabelle **mit Rahmen**, ausreichend hoch (ca. 10 Leerzeilen).

#### Layout
- Schriftart: Arial, 11pt (22 half-points)
- Seitenformat: A4, Ränder 2cm
- Kopfzeile: "Gymnasium Hofwil | DUY | [Gefäss] [Fachbereich]"
- Fusszeile: Seitenzahl zentriert
- Seitenumbruch vor grösseren Aufgabenblöcken, damit keine Aufgabe über zwei Seiten gebrochen wird, wenn vermeidbar.

### 7. Lösungen als versteckter Text

Die Lösungen werden **direkt in der Prüfung** als versteckter Text eingefügt (kein separates Lösungsdokument). Format:

- **Position**: In der rahmenlosen Antworttabelle, direkt unter der Frage.
- **Formatierung**: `vanish` (Word: versteckter Text), rote Schriftfarbe (`FF0000`).
- **Ein-/Ausblenden** in Word: Datei → Optionen → Anzeige → Ausgeblendeten Text anzeigen.
- **Layout-Neutralität**: Das Ein-/Ausblenden der Lösungen soll das Layout der Prüfung nicht wesentlich verändern. Die Tabellenhöhe muss ausreichend sein, um sowohl die Lösung (im eingeblendeten Zustand) als auch den Antwortplatz (im ausgeblendeten Zustand) aufzunehmen.

#### Inhalt der Lösungen
Jede Lösung enthält:
1. **Erwartete Antwort**: Konkret und vollständig formuliert.
2. **Punktevergabe**: Wie werden die Teilpunkte vergeben? z.B. "1P für korrekte Definition, 1P für passendes Beispiel".
3. **Taxonomiestufe**: z.B. "Taxonomie: K2 (Verstehen) / K3 (Anwenden)".
4. **Hinweise zur Bewertung**: Was wird auch akzeptiert? Häufige Fehler?
5. **Bei MC/R-F**: Korrekte Antwort + kurze Begründung, weshalb die anderen falsch sind.

### 8. Lehrplanbezug

Am Ende der Prüfung (ebenfalls als versteckter Text) angeben:
- Fach und Gefäss (z.B. SF W&R, Fachbereich VWL)
- Lehrplan 17 Kanton Bern, Zyklus und Semester
- Relevante Grobziele aus dem Lehrplan
- Taxonomieabdeckung: Welche Stufen (K1–K6) werden in welchen Aufgaben abgedeckt?

### 9. Technische Umsetzung (docx-js)

Für die Erstellung mit docx-js folgende Konventionen beachten:

```javascript
// Versteckter Text (Lösung)
new TextRun({ text: "Lösungstext", vanish: true, color: "FF0000", font: "Arial", size: 20 })

// Rahmenlose Tabelle für Antwortfeld
const noBorder = { style: BorderStyle.NONE, size: 0, color: "FFFFFF" };
const noBorders = { top: noBorder, bottom: noBorder, left: noBorder, right: noBorder };

// Tabelle mit Rahmen für Zeichenfelder
const border = { style: BorderStyle.SINGLE, size: 1, color: "000000" };
const borders = { top: border, bottom: border, left: border, right: border };

// MC-Checkbox: Unicode ☐ (\u2610)
new TextRun({ text: "A)  \u2610  Antworttext", font: "Arial", size: 22 })
```

### 10. Qualitätskontrolle vor Abgabe

Vor der Fertigstellung prüfen:
- [ ] Gesamtpunktzahl stimmt mit Summe der Teilpunkte überein
- [ ] Punktzahl entspricht ungefähr der Prüfungsdauer (1P/min)
- [ ] Mindestens 3 verschiedene Fragetypen verwendet
- [ ] Taxonomiestufen variiert (mindestens K1–K4)
- [ ] Lösungen für alle Aufgaben vorhanden
- [ ] Punktevergabe bei allen Lösungen angegeben
- [ ] Lehrplanbezug angegeben
- [ ] Bei Recht: Gesetzesartikel in Fragen und Lösungen referenziert
- [ ] Layout: Genügend Platz für Antworten
- [ ] docx validiert (validate.py)

## Beispiel-Interaktion

**Lehrer**: "Erstelle eine Prüfung für die 27abcd8f (GYM3, SF), Thema OR AT Vertragsabschluss, 45 Minuten, Hilfsmittel: OR/ZGB."

**Claude**: Erstellt eine 45-Punkte-Prüfung mit:
- Aufgabe 1: Richtig/Falsch zu Grundbegriffen (K1–K2, 6P)
- Aufgabe 2: MC zu Formvorschriften (K1–K2, 4P)
- Aufgabe 3: Fallstudie — Subsumption eines Vertragsabschlusses (K3–K5, 15P)
- Aufgabe 4: Lückentext zu Willensmängeln (K1, 5P)
- Aufgabe 5: Offene Frage — Vergleich zweier Vertragstypen (K4, 8P)
- Aufgabe 6: Stellungnahme — Beurteilung einer Vertragssituation (K5, 7P)

Alle Lösungen als versteckter Text integriert, mit Gesetzesartikeln, Punktevergabe und Taxonomiestufen.
